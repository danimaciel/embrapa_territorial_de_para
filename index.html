Como ler esta análise — metodologia

Para cada solução de inovação, o sistema compara o texto da solução (e, opcionalmente, o compromisso vinculado) com a base de Desafios de Inovação (DI) cadastrados, usando similaridade TF-IDF/cosseno. A coluna PARA traz o DI de maior similaridade (Top 1); quando esse DI sugerido difere do DI atualmente associado à solução, a Classificação indica o nível de confiança da divergência: Mantém DI atual (já é a melhor opção), Associação atual defensável (DI atual ainda é competitivo), Associação transversal/ambígua (tema permeia mais de um portfólio) ou Revisar associação (recomenda-se reclassificar). Use as abas acima para explorar por portfólio, por Desafio de Inovação específico, na lista completa ou nos indicadores agregados.

Cada faixa representa soluções migrando de um portfólio (esquerda · situação atual) para outro (direita · sugerido pelo TF-IDF). Largura proporcional ao volume. Clique em uma faixa para ver as soluções daquele fluxo.

Cada cartão é um Desafio de Inovação (DI) atual, com o portfólio ao qual pertence hoje. Os chips abaixo do texto mostram para quais DI as soluções vinculadas foram sugeridas (com a contagem de soluções). Clique em um chip para ver as soluções daquele par DI atual → DI sugerido.
